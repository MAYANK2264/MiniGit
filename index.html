MiniGit

A simplified version control system that mimics GitHub's core functionality, built with C++ backend for robust version control logic and React frontend for a stunning, modern user interface.

🚀 Get Started 🎥 Watch Demo
Live Demo
$ git clone https://github.com/MAYANK2264/MiniGit.git
$ cd MiniGit
$ ./scripts/dev.ps1
🚀 MiniGit is running on http://localhost:5174

Core Features

Repository Management

Initialize repositories, stage files, and create commits with ease. Full Git workflow support.

Branch Management

Create, switch, and merge branches. Handle conflicts with built-in resolution tools.

Beautiful UI

Modern, responsive interface with smooth animations. GitHub-inspired design with glassmorphism effects.

Commit History

Visualize commit history with interactive graphs. Navigate through repository states effortlessly.

Code Editor

Built-in code editor with syntax highlighting. View and edit files directly in the interface.

Docker Support

Easy deployment with Docker and Docker Compose. Run anywhere with containerized services.

Technology Stack

Backend (C++)

CMake build system
Custom HTTP server implementation
JSON-based storage for commits and metadata
SHA1-like hashing for commit identification
Dynamic Programming for diff calculations

Frontend (React)

Vite for fast development and building
TypeScript for type safety
Tailwind CSS for modern styling
Framer Motion for smooth animations
React Query for data fetching
Zustand for state management
